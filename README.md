### PySpark TF-IDF workshop

-----

#### Preparation process
1. Download&Install Anaconda Python 3.7 version.
https://www.anaconda.com/distribution/

2. Install PySpark.
You can follow these instructions - https://github.com/yajieli912/Spark_PySpark.

3. Clone this repo to Your machine.

4. Run "pyspark_check.ipynb". If it works without errors, congrats.

5. Download Airbnb data, "listings.csv" (it's listings summary) for the city that you like more from http://insideairbnb.com/get-the-data.html and move it in the folder "data".

6. Now You are ready to go through "tf-idf.ipynb".
